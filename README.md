## About The Project

An object-oriented programming project in C++ that was originally done during the period of 2017 Sep - 2019 Dec.

### Command Interpreter

Use I/O streams to create a robust parser UI to interpret user input.

### Classes

Creat a "back-end" for the resistive circuit simulator, which uses classes and dynamically allocated arrays to represent resistors and nodes in the circuit. 

### Linked Lists

Port the array based back-end to a linked list backed one, then had to solve for the voltages at every node in the circuit.

### Databases/Graphs

Implement a binary search tree from scratch to store a domain names and their corresponding IP address. This is called a "domain name server" and allowed for fast access to data based on an input string.

### Inheritance/Graphics

Write classes inheriting from a simple graphics framework (easyGL) and create a breadboard that allowed users to click, drag, and modify shapes on a screen.

### Built With

* C++
